# Version 1.0.0

After days of hard work, this brand-new UI is finally here!

## Installation Guide

### For Users

1. **Extract Certificate Info: Right-click the downloaded .msix installer and select Properties.**

2. **View Signature: Switch to the Digital Signatures tab, select the signer from the list (e.g., "Embedded Signature"), and click Details.**

3. **Install Certificate: In the pop-up window, click View Certificate, followed by Install Certificate.**

4. **Select Store Location: In the Certificate Import Wizard, choose Local Machine as the store location, then click Next.”**

5. **Specify Certificate Store: Select Place all certificates in the following store, click Browse, and choose Trusted Root Certification Authorities.**

6. **Finish Installation: Keep clicking Next until you reach Finish. Once the system confirms a successful import, you can double-click the .msix file to install the application normally.**
